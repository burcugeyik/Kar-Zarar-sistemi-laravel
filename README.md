# 📊 Kar-Zarar ve Stok Takip Sistemi

## 📖 Proje Hakkında

Kar-Zarar ve Stok Takip Sistemi, küçük ve orta ölçekli işletmelerin ürün, stok, satış, gider ve finansal süreçlerini tek bir platform üzerinden yönetebilmesi amacıyla Laravel kullanılarak geliştirilmiş web tabanlı bir yönetim uygulamasıdır.

Proje kapsamında kullanıcı kimlik doğrulama, ürün yönetimi, stok kontrolü, satış işlemleri, gider takibi ve fatura görüntüleme modülleri geliştirilmiştir.

---

## 🚀 Özellikler

* 🔐 Güvenli kullanıcı giriş sistemi
* 📦 Ürün ekleme, düzenleme ve silme
* 📊 Stok takip modülü
* 💰 Dinamik fiyat listesi
* 🛒 Satış oluşturma
* 📜 Satış geçmişi görüntüleme
* 💸 Gider kayıt sistemi
* 📈 Kar-Zarar takibi
* 🧾 Fatura görüntüleme
* 📱 Responsive kullanıcı arayüzü

---

## 🛠️ Kullanılan Teknolojiler

* PHP 8
* Laravel 12
* MySQL
* Blade Template Engine
* HTML5
* CSS3
* JavaScript
* Bootstrap
* Laravel Breeze

---

## 📂 Proje Yapısı

* Authentication
* Dashboard
* Product Management
* Stock Tracking
* Sales Management
* Expense Tracking
* Invoice System

---

## ⚙️ Kurulum

```bash
git clone https://github.com/burcugeyik/Kar-Zarar-sistemi-laravel.git

cd Kar-Zarar-sistemi-laravel

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
